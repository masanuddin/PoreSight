# PoreSight

This repo contains the packaged version of PoreSight.
Download and unzip `PoreSight.zip` to explore the source code.

PoreSight is an AI-powered computer vision project designed to analyze facial images and detect skin pores and imperfections using deep learning techniques.

The project explores how visual intelligence can be applied to personal skincare analysis in a non-invasive and data-driven way.

## Overview

Skin analysis is often subjective and inconsistent, relying heavily on manual observation or consumer-grade tools. PoreSight approaches this problem by using computer vision to extract visual patterns related to pores and skin texture from facial images.

The system focuses on clarity, reproducibility, and model-driven insight rather than cosmetic scoring.

The project is implemented using Python and modern computer vision libraries. Deep learning models are trained using established frameworks, with OpenCV supporting image processing and analysis.

## Purpose

This project demonstrates applied skills in computer vision, deep learning, and image-based analysis. PoreSight highlights how AI can be used responsibly for personal analysis and decision support in skincare-related applications.
